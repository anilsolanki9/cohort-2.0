# Resume

1. Personal Details

Dont put personal details (name age gender), Only

- Name
- Email
- Contact
- Linkedin, Github, Portfolio(Okaat Dikhaao Apni), Leetcode
- Designation (eg. Full Stack Developer)

2. Experirnce

- Company name, working period, What you done (To achieve x, you did y, and got z results)

3. Projects (Best 2-3 Projects) (Must have Good code, architecture, system design, Best practices). Live Link,

- project -> Name, Tech stack, Live link, Short Description (what problem you solved) max 3 lines.

4. Skills

- Never rate your skills, React **\*** etc, Dont give your stars.
- Never write what you dont know.
- Languages, Tools, Technologies,

5. Achievements / Certifications

- No Hoobbies
- Everything you write must have proof. Every word should be proved by work of document.
- Real and impactfull Acheievments.

- Never do
- Grammer mistakes
- Highlighting by more then one color.
- Putting photo on resume.
- Same resume for every company
- Projects me portfolio mat daaalo.
