# Day 56 - Advance JS (OOPs in JavaScript)

This directory contains learning materials and code related to **Object-Oriented Programming (OOP)** in JavaScript.

## Files Structure

- **`Day56_OOPs in JS.md`**: Detailed notes on OOP concepts in JavaScript, including objects, classes, constructors, prototypes, and inheritance with code examples.
- **`index.html`**: A basic HTML boilerplate to run and test the JavaScript code in the browser.
- **`script.js`**: An active JavaScript file for practicing OOP concepts like creating classes and writing constructors.

## Concepts Covered
- JavaScript Objects
- Classes and Blueprinting
- Constructors
- Prototypes and Method Sharing
- Inheritance Basics
